# Deep-Learning

COMP3340 Applied Deep Learning — The University of Hong Kong.

## Contents

`MLP_CNN_Transformer_Diffusion.ipynb` — Implements four deep learning architectures from scratch in PyTorch:

- **MLP** — Multi-layer perceptron classifier
- **CNN** — Convolutional neural network image classifier
- **Transformer** — Attention-based sequence model
- **Diffusion** — Denoising diffusion probabilistic model (training and sampling)

## Setup

pip install torch torchvision matplotlib numpy
